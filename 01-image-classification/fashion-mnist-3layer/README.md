# Fashion-MNIST 三层神经网络

**来源仓库**：`cxk049/computer-vision`（已归档）

## 项目简介

从零开始构建三层神经网络分类器，在 [Fashion-MNIST](https://github.com/zalandoresearch/fashion-mnist) 数据集上训练以实现图像分类。

## 文件结构

```
fashion-mnist-3layer/
├── homework.ipynb         # 主 Notebook（数据加载、训练、可视化）
├── trained_model.pkl      # 训练好的模型权重
└── HOMEWORK_ORIGINAL.txt  # 原任务描述（保留原文）
```

## 使用说明

打开 `homework.ipynb`，按顺序执行所有 cell。Notebook 中已包含：
- 数据集加载
- 模型定义
- 训练流程
- 测试与可视化

> 注：训练好的模型权重 `trained_model.pkl` 已上传，可直接加载进行测试，无需重新训练。
